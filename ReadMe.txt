これはGithubというプログラムを公開したりバージョン管理することによく使われているサービスです。最近では本の執筆やWebデザインのパーツのデザインのバージョン管理や共同制作にも使われているそうです。

基本的に非公開にするのは個人的なポリシーに反するので、誰でも見ることができる状態になっています。Githubのアカウントを持っていればForkという機能で他の人もその文書を編集することができますが、Forkしたよという情報は制作者にちゃんと伝わるので、勝手に編集される危険性はありません。
